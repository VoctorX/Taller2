3. Listas como Pilas (Stack)

Implementaci√≥n completa de una pila usando una lista, siguiendo el principio LIFO (Last In, First Out).

class Pila:
    def __init__(self):
        self.items = []
    
    def apilar(self, elemento):
        self.items.append(elemento)
    
    def desapilar(self):
        if self.esta_vacia():
            return None
        return self.items.pop()
    
    def ver_tope(self):
        if self.esta_vacia():
            return None
        return self.items[-1]
    
    def esta_vacia(self):
        return not self.items

4. Listas como Colas (Queue)

Uso de collections.deque para implementar una cola eficiente con principio FIFO (First In, First Out).

from collections import deque
import time

class SistemaPedidos:
    def __init__(self):
        self.pendientes = deque()  # Cola de pedidos pendientes
        self.completados = []     # Lista de pedidos completados
        self.numero_pedido = 1
    
    def agregar_pedido(self, cliente, items):
        pedido = {
            'numero': self.numero_pedido,
            'cliente': cliente,
            'items': items,
            'hora': time.strftime("%H:%M:%S"),
            'estado': 'pendiente'
        }
        self.pendientes.append(pedido)
        self.numero_pedido += 1
        return pedido['numero']

5. Comprensi√≥n de Listas

Creaci√≥n concisa de listas usando sintaxis compacta para filtrado y transformaci√≥n.

# Filtrar y transformar temperaturas
celsius = [25, 30, 15, 35, 10, 20]

# Solo temperaturas mayores a 20¬∞C, convertidas a Fahrenheit
fahrenheit = [(c * 9/5) + 32 for c in celsius if c > 20]
print(fahrenheit)  # [77.0, 86.0, 95.0]

Operaciones comunes:

  • Filtrado: [x for x in lista if condicion]
  • Transformaci√≥n: [expresion for x in lista]
  • Combinado: [expresion for x in lista if condicion]
